you avoid bugs and write cleaner code.

**Key concepts:**

- **Global scope** - Variables declared outside any function are accessible
  everywhere
- **Local/Function scope** - Variables declared inside a function are only
  accessible within that function
- **Block scope** - Variables declared with `let` and `const` inside `{}` are
  only accessible within those braces
