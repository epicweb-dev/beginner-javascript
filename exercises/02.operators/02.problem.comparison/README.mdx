# Comparison Operators

ğŸ‘¨â€ğŸ’¼ Programs need to compare values to make decisions. Comparison operators
return `true` or `false` based on how values relate to each other.

The comparison operators are:

- `===` Strict equality (same value AND same type)
- `!==` Strict inequality (not equal)
- `<` Less than
- `>` Greater than
- `<=` Less than or equal
- `>=` Greater than or equal

<callout-warning>
	Always use `===` (strict equality), not `==` (loose equality). Loose equality
	does type coercion which can lead to unexpected results!
</callout-warning>
