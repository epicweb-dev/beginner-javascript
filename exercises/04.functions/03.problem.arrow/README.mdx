functions. They're especially useful for short, simple functions.

The syntax:

```js
// Traditional function
function add(a, b) {
	return a + b
}

// Arrow function
const add = (a, b) => {
	return a + b
}

// Even shorter for single expressions
const add = (a, b) => a + b
```
