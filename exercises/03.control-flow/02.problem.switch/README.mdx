# Switch Statements

ğŸ‘¨â€ğŸ’¼ When you're comparing one value against many specific possibilities, a
`switch` statement is cleaner than many if/else statements.

The syntax:

```js
switch (value) {
	case 'option1':
		// code for option1
		break
	case 'option2':
		// code for option2
		break
	default:
		// code if no case matches
}
```
