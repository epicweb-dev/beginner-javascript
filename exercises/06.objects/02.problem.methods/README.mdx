we call it a **method**.

```js
const calculator = {
	add: function (a, b) {
		return a + b
	},
	// Shorthand syntax (preferred):
	subtract(a, b) {
		return a - b
	},
}

calculator.add(5, 3) // 8
calculator.subtract(5, 3) // 2
```
