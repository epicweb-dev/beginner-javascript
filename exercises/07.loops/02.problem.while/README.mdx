# While Loops

ğŸ‘¨â€ğŸ’¼ When you don't know exactly how many iterations you need, `while` loops
continue running as long as a condition is true.

The syntax:

```js
while (condition) {
	// code to repeat
}
```

For example:

```js
let count = 0
while (count < 3) {
	console.log(count)
	count++
}
// Logs: 0, 1, 2
```
