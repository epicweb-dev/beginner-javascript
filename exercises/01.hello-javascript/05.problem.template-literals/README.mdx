# Template Literals

ğŸ‘¨â€ğŸ’¼ Template literals are a powerful way to work with strings in JavaScript.
Instead of using quotes (`'` or `"`), you use backticks (`` ` ``).

The magic of template literals is **string interpolation** - you can embed
variables and expressions directly inside the string using `${...}`:

```js
const name = 'Alice'
const greeting = `Hello, ${name}!`
// "Hello, Alice!"
```

You can also create **multi-line strings** without any special characters:

```js
const poem = `Roses are red,
Violets are blue,
JavaScript is awesome,
And so are you!`
```
