<callout-info>
JavaScript uses "short-circuit evaluation": - With `&&`, if the first value is
false, JavaScript doesn't check the second - With `||`, if the first value is
true, JavaScript doesn't check the second

This becomes useful for patterns like `user && user.name` to safely access
properties.
</callout-info>
