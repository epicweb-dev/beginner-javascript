# For...of Loops

ğŸ‘¨â€ğŸ’¼ When you want to iterate over an array and just need each value (not the
index), `for...of` provides clean, readable syntax.

The syntax:

```js
for (const item of array) {
	// use item
}
```

For example:

```js
const fruits = ['apple', 'banana', 'cherry']
for (const fruit of fruits) {
	console.log(fruit)
}
// Logs: apple, banana, cherry
```
