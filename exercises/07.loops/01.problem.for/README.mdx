# For Loops

ğŸ‘¨â€ğŸ’¼ When you need to repeat code a specific number of times, the `for` loop is
your tool of choice.

The syntax:

```js
for (initialization; condition; update) {
	// code to repeat
}
```

For example:

```js
for (let i = 0; i < 5; i++) {
	console.log(i) // Logs 0, 1, 2, 3, 4
}
```

The three parts:

1. **Initialization** (`let i = 0`) - Runs once before the loop starts
2. **Condition** (`i < 5`) - Checked before each iteration; loop stops when false
3. **Update** (`i++`) - Runs after each iteration
