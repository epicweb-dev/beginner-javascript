Creating an array:

```js
const fruits = ['apple', 'banana', 'cherry']
```

Accessing elements (zero-indexed):

```js
console.log(fruits[0]) // "apple" (first element)
console.log(fruits[1]) // "banana" (second element)
console.log(fruits[2]) // "cherry" (third element)
```
